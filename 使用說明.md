# 🔐 萬用加密工具 — 完整安裝與使用教學

> 寫給完全不熟悉電腦操作的人。照著本文從頭到尾做一次，之後每次只要「雙擊開啟程式」即可。

---

## 這個工具能做什麼？

- 🔒 把 **PDF、Word、Excel、PowerPoint、圖片（jpg/png…）、或任何檔案** 加上密碼保護
- 🖼️ 把多張掃描圖片（身分證、收據、單據）合併成一份加密 PDF
- 🔑 自動產生高強度密碼、檢查密碼強不強
- 📂 加密完成後可自訂輸出資料夾位置
- 全程 **不連網路**，所有處理都在你自己的電腦上完成

> 加密後的檔案要怎麼打開？不需要另外的解密工具 —— 用一般的解壓縮軟體
> （或系統內建解壓縮）、PDF 閱讀器、或 Office，輸入密碼即可直接打開。

---

## 第一步：安裝 Python

如果你的電腦裡從沒安裝過 Python，請先做這一步（做過的人可以跳到「第二步」）。

1. 打開瀏覽器，前往：**https://www.python.org/downloads/**
2. 點選黃色的「Download Python」按鈕，下載安裝檔。
3. 打開下載好的安裝檔，**畫面最下方務必勾選「Add Python to PATH」**（這一步很重要，沒勾會導致後面指令找不到 Python）。
4. 點選「Install Now」，等待安裝完成。

安裝完成後，驗證是否成功：

- **Windows**：按下鍵盤 `Win` 鍵，輸入「命令提示字元」或「PowerShell」並打開。
- **Mac**：打開「應用程式」→「工具程式」→「終端機」。

在打開的黑色視窗裡輸入以下文字，然後按 Enter：

```
python --version
```

如果畫面顯示類似 `Python 3.12.3` 的文字，代表安裝成功。
（Mac 使用者如果顯示找不到指令，請改試 `python3 --version`）

---

## 第二步：安裝這個工具需要的套件

同樣在剛才打開的黑色視窗（終端機）裡，輸入以下指令並按 Enter（會需要幾分鐘下載）：

```
pip install customtkinter pypdf cryptography pyzipper pillow
```

看到最後出現 `Successfully installed...` 字樣就代表成功了。

> 這五個套件的作用：
> - `customtkinter`：整個圖形介面（視窗、按鈕、卡片外觀）都靠它
> - `pypdf` + `cryptography`：負責 PDF 加密
> - `pyzipper`：負責「通用高強度加密」（適用任何檔案類型）
> - `pillow`：負責圖片處理、圖片轉PDF功能

### 這個套件是「選用」的，看你需不需要：

**如果你想要用「Word / Excel / PowerPoint 原生加密」**（打開檔案時直接跳出密碼視窗，效果等同你在 Office 裡自己設密碼）：
- 僅限 **Windows** 電腦，且電腦裡必須已經安裝**正版 Microsoft Office**
- 安裝指令：
```
pip install pywin32
```
- 如果你沒有正版 Office，或使用 Mac，**不用安裝這個**，直接使用「通用高強度加密」即可，安全性一樣很高。

## 第三步：開啟工具

1. 把 `萬用加密工具.py` 這個檔案放到你隨便一個好記的資料夾（例如桌面）。
2. **方法A（最簡單）**：直接對這個檔案「滑鼠右鍵」→「開啟檔案」→ 選擇 Python，或直接雙擊它。
3. **方法B（如果方法A沒反應）**：打開終端機，輸入：
   ```
   cd 桌面所在的資料夾路徑
   python 萬用加密工具.py
   ```
   例如：
   ```
   cd Desktop
   python 萬用加密工具.py
   ```

程式視窗打開後，你會看到最上方有一排小圖示顯示目前哪些功能可以用（綠色勾勾 ✅ 代表可以用，紅色 ⛔ 代表缺少套件）。

---

## 第四步：開始使用（畫面會用 5 個步驟引導你）

打開程式後，點選左側「🔒 加密檔案」，畫面上方會有一條步驟指示條，
跟著走一次就完成了：

1. **① 選擇檔案** — 按「選擇檔案」或「選擇資料夾」，挑出你要保護的內容。
2. **② 加密方式** — 第一次使用建議選「🌍 通用高強度加密」，幾乎適用所有情況。
3. **③ 設定密碼** — 輸入兩次密碼，或按「🎲 自動產生高強度密碼」（記得先複製保存）。
4. **④ 輸出位置** — 預設存在原始檔案旁邊；如果想另外指定資料夾，選「另外指定一個資料夾」再按「選擇資料夾」。
5. **⑤ 開始加密** — 確認上面顯示的設定摘要沒問題後，按「🔒 開始加密」，
   完成後會自動打開輸出資料夾。

每個步驟都可以按「‹ 上一步」回去修改，不用擔心設定錯誤。

> ⚠️ **重要：如果你選「🌍 通用高強度加密」**，產生的 `.zip` 檔案在 Windows
> 上會顯示成「壓縮的資料夾」圖示，但 **Windows 內建的「解壓縮全部」無法
> 打開它**（Windows 內建功能只認識較舊、較不安全的加密方式）。
> 請先安裝免費的 **7-Zip**（<https://www.7-zip.org/>），下載安裝後，對著
> `.zip` 檔按滑鼠右鍵 →「7-Zip」→「開啟壓縮檔」，輸入密碼即可正常使用。
> 如果不想額外安裝軟體，改選「📕 PDF 原生加密」或「📘 Office 原生加密」，
> 這兩種打開時完全不需要額外工具。

## 常見狀況排解

| 狀況 | 解法 |
|---|---|
| 打開終端機輸入 `python --version` 顯示「找不到指令」 | 回到第一步，重新安裝 Python，安裝時務必勾選「Add Python to PATH」 |
| 輸入 `pip install ...` 顯示「找不到指令」 | 改用 `python -m pip install ...` 試試看 |
| 雙擊 .py 檔案沒有反應 | 改用終端機的「方法B」開啟 |
| 「Word/Excel/PowerPoint 原生加密」選項是灰色 | 代表沒偵測到 Office 或不是 Windows，改用「通用高強度加密」即可，效果一樣安全 |
| 忘記密碼 | 非常抱歉，這是無法救回的（這也是高強度加密的特性），請務必事先妥善保存密碼 |
| **加密後的 .zip 檔看起來像資料夾、打不開、顯示無法使用** | 這是 Windows 內建解壓縮不支援 AES-256 加密造成的，**不是檔案壞掉**。安裝 7-Zip（<https://www.7-zip.org/>）後用它開啟即可，見上方⚠️說明 |
| 收到加密檔案的人不知道怎麼打開 | .zip 檔請對方安裝 7-Zip 或 WinRAR 再輸入密碼；.pdf 或 Office 檔直接雙擊打開，跳出密碼視窗時輸入密碼即可，不需要安裝任何額外工具 |
| 畫面文字模糊 | 程式已內建 Windows 高解析度螢幕修正，若安裝的是舊版本檔案，請改用最新版本 |

## 安全性說明

- 通用加密方式使用 **AES-256** 演算法加密，這是目前業界（包含銀行、政府機關）廣泛採用的高強度加密標準，理論上以現有運算能力破解需要極長時間。
- PDF 原生加密同樣採用 AES-256。
- 密碼強度才是真正的關鍵：再強的加密演算法，如果密碼是 `123456` 這種容易猜到的密碼，保護力都會大幅下降。建議使用工具內建的「密碼產生器」。
- 本工具不會將任何檔案內容或密碼傳送到網路上，所有運算都在你自己的電腦本機完成。
